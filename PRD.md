# PRD: 네이버 카페 IT 교육 게시글 모니터링 시스템

**문서 버전:** 1.0
**작성일:** 2026-03-20
**작성자:** 인터뷰 기반 작성

---

## 1. 배경 및 문제 정의

### 1.1 현재 상황
IT 교육 영업 담당자가 네이버 카페(ak573)의 4개 게시판을 **10분마다 수동으로 검색**하여 IT 교육 관련 문의 게시글을 탐지하고, 해당 게시글에 댓글을 달고 이메일로 제안서를 발송하는 업무를 반복하고 있다.

### 1.2 핵심 문제
| 문제 | 영향 |
|------|------|
| 10분 주기 수동 검색 | 업무 집중력 분산, 반복 노동 |
| 게시글 탐지 누락 가능성 | 영업 기회 손실 (경쟁자에게 선점 당함) |
| 제안서 작성 30분/건 | 하루 1~1.5시간 소요 |

### 1.3 비즈니스 맥락
- **경쟁 강도:** 동일 카페 내 다수의 경쟁자 존재 → **속도가 곧 경쟁력**
- **전환율:** 게시글 발견 → 최종 계약 약 5%
- **처리량:** 하루 평균 2~3건 답변/제안서 발송

## 2. 목표

### 2.1 핵심 목표
> **게시글 탐지 자동화를 통해 영업 반복 업무를 제거하고, 경쟁자보다 빠르게 고객에게 접근한다.**

### 2.2 성공 지표 (6개월 후)
| 지표 | 현재 | 목표 |
|------|------|------|
| 게시글 탐지 소요 시간 | 수동 10분 주기 검색 | 0분 (완전 자동) |
| 탐지 누락률 | 알 수 없음 | 0% |
| 영업 반응 속도 | 최대 10분 지연 | 10분 이내 알림 수신 |
| 계약 건수 | 현재 대비 | 증가 (정량 목표 추후 설정) |

## 3. 사용자 정의

### 3.1 주 사용자
- **IT 교육 영업 담당자 (1인)**
- 팀이 있으나 이 시스템은 담당자 개인용으로 운영
- 기술적 유지보수 능력 없음 → 시스템은 자동으로 안정 운영되어야 함

### 3.2 최종 고객 (간접 대상)
- 네이버 카페에 IT 교육을 문의하는 기업 담당자
- 업종/규모 제한 없음 (대기업, 중소기업, 공공기관, 스타트업 등)

## 4. 기능 요구사항

### 4.1 [P0] 게시글 자동 탐지 — 핵심 기능

#### 모니터링 대상
| 게시판 | 메뉴 ID |
|--------|---------|
| 교육훈련 게시판 | 530 |
| 교육기획 게시판 | 14 |
| 강사추천 게시판 | 334 |
| 자유게시판 | 3 |

#### 탐지 조건
- 게시글 제목 또는 본문에 IT 교육 관련 키워드 포함 시 탐지
- 주요 키워드: AI, 인공지능, 바이브코딩, AI 에이전트, ChatGPT, 챗GPT, LLM, 생성형 AI, 머신러닝, 딥러닝, 프롬프트, 데이터사이언스, 파이썬, 자동화, RPA, 디지털전환, 코딩교육 등

#### 탐지 주기
- **10분 간격** 자동 스캔
- PC 부팅 시 자동 시작

#### 중복 방지
- 이미 탐지된 게시글은 재알림 없음 (SQLite DB 기반 관리)

---

### 4.2 [제외] 이메일 알림 — 대시보드로 대체

> **결정:** 웹 대시보드에서 실시간 탐지 현황 및 미응대 게시글을 즉시 확인할 수 있으므로 이메일 알림 기능은 불필요하다고 판단, 제외함.

---

### 4.3 [P0] GitHub 자동 저장

#### 저장 대상
- 탐지된 모든 IT 교육 관련 게시글

#### 저장 형식
- Markdown 파일
- 경로: `posts/{게시판명}/{YYYY-MM}/{게시글ID}_{제목}.md`

#### 저장 내용
- 게시판명, 작성자, 작성일, 원문 URL, 탐지 키워드, 본문 전문

---

### 4.4 [P1] 안정적 자동 운영

- PC 재부팅 후 자동 재시작 (Windows 시작프로그램 등록)
- 오류 발생 시 로그 기록 (`monitor.log`)
- 네이버 쿠키 만료 시 담당자가 수동으로 갱신 (약 30일 주기)

## 6. 현재 수동 유지 영역 (자동화 제외)

아래 단계는 담당자가 직접 수행하며, 이 시스템의 범위 밖이다.

| 단계 | 방식 |
|------|------|
| 댓글 작성 | 직접 수동 작성 |
| 이메일 수집 | 게시글 내 기재 또는 쪽지로 문의 |
| 제안서 작성 | 건별 맞춤 작성 (회사소개, 강사 프로필, 커리큘럼, 비용, 일정) |
| 이메일 발송 | 직접 발송 |
